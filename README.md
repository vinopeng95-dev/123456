# 🎬 P級電影推薦系統

查詢本週上映的保護級(P)電影，適合6歲以上觀賞。

## 功能特點

- ✅ 查詢本週上映的 P 級（保護級）電影
- ✅ 支援查詢其他分級（G、PG12、PG15、R）
- ✅ 顯示電影海報、上映日期
- ✅ 提供詳細介紹頁超連結
- ✅ 響應式設計，手機電腦都能看

## 台灣電影分級制度

| 分級 | 名稱 | 說明 |
|------|------|------|
| P | 保護級 | 6歲以上才能觀看 |
| G | 普遍級 | 所有人都可以看 |
| PG12 | 輔導12級 | 12歲以上須家長陪同 |
| PG15 | 輔導15級 | 15歲以上須家長陪同 |
| R | 限制級 | 18歲以上才能觀看 |
